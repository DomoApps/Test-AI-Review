## Get File By ID 

**Method**: `GET`  
**Endpoint**: `/data/v1/data-files/<FILE_ID>`

**Example**:

```json http
{
  "method": "GET",
  "url": "https://{instance}.domo.com/data/v1/data-files/<FILE_ID>",
  "headers": {
    "X-DOMO-Developer-Token": "",
    "Content-Type": "application/json"
  },
  "body": {}
}
```

**Response**:  

```json
200:
{}

```
